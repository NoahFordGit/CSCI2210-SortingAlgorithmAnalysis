# Sorting Algorithm Analysis

This project implements and benchmarks several common sorting algorithms. The primary goal is to analyze their performance across different input sizes and compare their efficiency through measured runtime data and written evaluation.

## Overview

- Implementations of multiple sorting algorithms  
- Benchmarking framework to measure and compare performance  
- Collected runtime data for analysis  
- A written document describing and comparing the algorithms  

## Purpose

The objective of this project is to:

- Understand how various sorting algorithms work  
- Evaluate their real-world performance  
- Compare time complexity behavior with empirical results  
- Identify strengths, weaknesses, and practical trade-offs  

## Usage

1. Build the project using your preferred C++ compiler or IDE  
2. Run the program to execute the benchmarking tests  
3. Review the output results  
4. Refer to the analysis document for detailed comparisons  

## Deliverables

- Sorting algorithm implementations  
- Benchmark results  
- Comparative analysis write-up
